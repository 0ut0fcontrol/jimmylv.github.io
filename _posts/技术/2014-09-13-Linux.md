##学习笔记
###查看内核版本
```
uname -r
```
稳定版本的偶数版，如2.6.x，适合于商业与家用环境使用；开发中版本，如2.5.x，适合开发特殊功能的环境。

##实际问题
###登陆root

- Mac：（若只以普通用户身份登陆，将root改为username即可）

```
ssh root@IPaddress
```
- Linux：

```
su -
```

###建立网络映射
- Mac：`Finder`->`前往`->`连接服务器`->输入`smb://IPaddress/samba`->`连接`
- Linux：`位置`->`连接服务器`->“服务类型”选择`自定义位置`->输入`smb://IPaddress/samba`->`连接`
